# 通用Linux自动化工具集

## 工具简介

此工具为纯脚本自动化工具，适用于各种 Linux 发行版与主流面板，提供系统查询、证书管理、网络检查、系统分析和文件管理。

## 版本信息

- 当前版本: v2.1
- 更新日期: 2026-03-20
- 版本历史:
  - v2.1: 修复菜单语法、去除 emoji、增强端口占用进程信息和高级进程杀掉交互、自动证书路径搜索
  - v2.0: 增加版本化管理、抓包管理功能
  - v1.0: 初始版本

## 核心功能

- 系统信息查询
- SSL 证书管理（自动搜索面板/系统路径）
- 网络工具（端口占用 PID 查询、杀进程双重确认）
- 系统工具（资源、进程、日志）
- 文件管理（搜索、权限、统计）

## 使用方法

1. 下载脚本:

```bash
wget -O linux-tools.sh "https://raw.githubusercontent.com/hhtbing/public-data/main/linux-tools/linux-tools.sh"
chmod +x linux-tools.sh
```
# 🚀 通用Linux自动化工具集

## 📋 工具简介

这是一个功能强大的通用Linux自动化工具集，适用于各种Linux系统和面板，提供了丰富的系统管理功能。

## 📌 版本信息

- **当前版本**: v2.1
- **更新日期**: 2026-03-20
- **版本历史**:
  - v2.1: 菜单/子菜单详细中文说明，所有子菜单支持回退，emoji风格，功能精细化
  - v2.0: 增加版本化管理、抓包管理功能
  - v1.0: 初始版本，包含系统信息、SSL管理、网络工具、系统工具、文件管理

## ✨ 核心功能

- 🖥️ **系统信息查询**：显示主机、CPU、内存、磁盘、网络等详细信息
- 🔒 **SSL证书管理**：适配多面板，支持部署、续期、SAN、通配符、覆盖等
- 📡 **网络工具**：端口占用、进程管理、路由、Ping等，支持交互杀进程
- 🛠️ **系统工具**：资源监控、进程、磁盘、日志等
- 📁 **文件管理**：文件搜索、内容查找、权限、统计等
- 🕵️ **抓包管理**：网口选择、关键词过滤、导入导出、分析等
- 🔑 **权限管理**：sudo/root切换，安全提示

## 🧭 菜单与操作说明

所有菜单和子菜单均支持输入 `b` 或 `返回` 回退主菜单。

### 主菜单
```
==========================================
  🚀 通用Linux自动化工具集 (v2.1)
==========================================
  1. 查询系统信息（显示主机、CPU、内存、磁盘、网络等详细信息）
  2. SSL证书管理（适配多面板，支持部署、续期、SAN、通配符等）
  3. 网络工具（端口占用、进程管理、路由、Ping等，支持交互杀进程）
  4. 系统工具（资源监控、进程、磁盘、日志等）
  5. 文件管理（文件搜索、内容查找、权限、统计等）
  6. 抓包管理（网口选择、关键词过滤、导入导出、分析等）
  7. 权限管理（sudo/root切换，安全提示）
  0. 退出
  输入 b 或 返回 可随时回退主菜单。
```

### SSL证书管理
- 1. 查看当前证书状态（自动适配多面板路径）
- 2. 按域名搜索并部署证书（输入域名，自动查找并可一键部署）
- 3. 全局搜索系统中所有证书（自动遍历常见面板和路径）
- 4. 手动指定证书路径（输入证书和私钥路径，校验并部署）
- 5. 跨域名证书部署（支持多个域名一键申请和部署）
- 6. 查询证书覆盖情况（批量输入域名，检查哪些已覆盖）
- 7. 交互式申请SAN证书（多域名证书，交互输入）
- 8. 交互式申请通配符证书（如*.example.com，需DNS验证）
- 9. 域名证书自动延期（自动续期，支持certbot/acme.sh）
- 10. 域名证书手动延期（手动续期指定域名）
- 11. 查询证书延期状态（查看所有证书续期状态）
- 0. 返回主菜单

### 网络工具
- 1. 网络连接状态（显示当前主机所有监听端口和连接）
- 2. 端口占用情况（可查进程并交互杀进程，支持倒计时和二次确认）
- 3. Ping 测试（输入目标IP或域名，测试连通性）
- 4. 路由表查看（显示主机路由表）
- 0. 返回主菜单

### 系统工具
- 1. 系统资源监控（top命令，显示CPU/内存/负载等）
- 2. 进程管理（按CPU排序，显示前20个进程）
- 3. 磁盘空间分析（根目录各子目录空间占用）
- 4. 系统日志查看（最新50行日志）
- 0. 返回主菜单

### 文件管理
- 1. 文件搜索（支持通配符，指定目录）
- 2. 文本内容搜索（递归查找目录下所有文件内容）
- 3. 文件权限管理（显示并修改文件权限）
- 4. 文件大小统计（目录下文件/子目录大小排序）
- 0. 返回主菜单

### 抓包管理
- 1. 开始抓包（选择网口、工具、过滤条件、保存文件名）
- 2. 分析已保存的包（统计、内容、协议过滤、关键词搜索）
- 3. 导入包文件（支持pcap/pcapng）
- 4. 导出包文件（导出到指定路径）
- 5. 抓包工具设置（检测/安装tcpdump、tshark等）
- 0. 返回主菜单

### 权限管理
- 1. 切换到管理员权限 (sudo)
- 2. 切换回普通用户权限
- 0. 返回主菜单
